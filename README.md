# 📸 시흥시 오이도 랜드마크 이미지 인식 웹어플리케이션
`Google Teachable Machine` 의 이미지 인식을 활용하여 오이도 랜드마크의 정보를 확인하고 퍼즐을 환성할 수 있는 웹어플리케이션

오이도를 관광하며 특정 랜드마크를 웹캠으로 비추어 관련 정보 제공

정보 확인 이후 퍼즐을 하나씩 맞추어 최종적으로 퍼즐을 완성

## 📅 제작 기간
